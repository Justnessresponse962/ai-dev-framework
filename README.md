## 🖥️ What this app is for

ai-dev-framework helps you manage AI-assisted software work with a spec-first process. It gives you a clear path from idea to finished result. It is built for people who want structure, checks, and steady progress while using Claude Code.

Use it when you want to:

- Define a task before work begins
- Keep changes tied to a clear spec
- Check work before moving to the next step
- Reduce confusion during AI-assisted development
- Keep software work organized from start to finish

## 🛠️ Typical workflow

A normal flow with this framework looks like this:

1. Start with a short idea.
2. Turn the idea into a spec.
3. Review the spec for clear goals.
4. Split the work into small tasks.
5. Use Claude Code to help with each task.
6. Check the output after each step.
7. Fix any issues before moving ahead.
8. Mark the task as done when it meets your rules.

This keeps the work in order and helps you avoid missing steps.
